Реализовать многостраничное web-приложение, 
состостоящее минимум из 3 страниц (landing можно). 

Минимальный стек технологий для приложения: 
Nginx/Apache2(уже настраивали), 
Http, 
JSON, 
HTML (обязательно наличие элементов управления в виде кнопок, элементов ввода), 
CSS (можно использовать фреймворки Bootstrap/Foundation), 
Javascript (любые библиотеки, типа JQuery, Select2, Bootstrap и тд...), 
Ajax. 

Обязательно использование пакетного менеджера (npm/yarn/bower/...) и 
сборщика (webpack/rollup/parcel/...). 

Залить код на github.com.
